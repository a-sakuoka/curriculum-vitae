### システムリプレイス【Hyper-V/WindowsServer/Linux】(2019年11月-12月（2ヶ月）)<br>
【プロジェクト概要】リース満了に伴い、基幹サーバシステムの老朽化更新<br>
［担当業務］基本設計、詳細設計、環境構築、テスト、運用支援（※ 一人で実施）<br>
- サーバリプレイス案件でNEC製の機種(1台)にホストOS4台を構築し、その中で仮想ゲストOSを複数台構築する。<br>
ゲストOSは基本レプリケーション構成でホスト障害に備えて、アクティブサーバとレプリカサーバで構成する。<br>
- [Hyper-V構成]<br>
WindowsServer2016 Std(ホストOS:4台、ゲストOS:8台)<br>
[ホストA(アクティブ) - ホストB(レプリカ)]、　[ホストC(アクティブ) - ホストD(レプリカ)]上記組み合わせにより、アクティブ側のホストに運用稼働するゲストOSを搭載する。<br>
レプリカ側のホストには運用ゲストのレプリカサーバを配置するため、普段は未起動。<br>
障害時にはレプリケーションしてレプリカサーバで稼働する仕様。<br>
ミドル：以下組み合わせによるLinuxHA環境　OracleDatabae12c、RDS(リモートデスクトップ管理)　DNS、AD(ドメイン管理)、CitrixXenApp
